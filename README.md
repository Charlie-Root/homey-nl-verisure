# Verisure

Add support for verisure alarm.

### Features:
* Alarm state (armed/disarmed)
* Alarm state in flow
* Temperature readings

* Humidity readings
* Door/Window sensors

Author: Wieger Bontekoe - SKYNET ICT B.V.
